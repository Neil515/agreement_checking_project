# 📘 README：AI 合約風險條款摘要工具助手

這是一個幫助大家快速看應「合約」或「網站服務條款」裏面藏有什麼風險的小工具。

很多人每天都會點「我同意」，但實際上根本沒時間看完那些密密鬼鬼的條款。這個工具的目的是： 👉 幫你找出裡面可能會讓你吃虧的內容，並用簡單的話解釋給你聽。

---

## 🧩 這個工具是做什麼用的？

* 讀取條款（例如網站的使用者協議、隱私政策）
* 幫你切成一條一條的句子
* 針對每一條條款判斷「是否有風險」，並標出「風險等級」：高、中、低
* 最後產出一份簡單易懂的摘要報告

---

## ⚡️ 多語言支援

未來計畫中，APP 會支援同時分析中文與英文條款，不需分成兩個 APP：

* 語言判別後，自動套用對應語系 Prompt
* UI 上可顯示正在分析的條款語言
* 使用者不需瞭解技術能力，簡潔一致的使用點擊頁面

## 🔧 開發流程與經營計畫

### 建議開發順序：「順序分階 + 模組獨立 + 可測試」

| 階段    | 內容                          | 總觀目的          |
| ----- | --------------------------- | ------------- |
| 分句    | 分析條款内容、切成獨立句子               | 便於 AI 對應分析    |
| 前處理   | 語言判斷、資料清潔                   | 確保 AI 可正確認識條款 |
| AI 分析 | 套用 Prompt，判別風險、類型、理由        | 根據標準說明輸出      |
| 結果展示  | 合成分析結果，輸出 JSON 或 HTML 展示    | 產出用戶看得懂的總算    |
| 用戶回饋  | 筆記「正確 / 錯誤」判斷，優化 Prompt 或模型 | 改進模型正確率       |

---

### 📃 各階段所需資源詳表

| 階段     | 程式/檔案                                                    | 說明                        |
| ------ | -------------------------------------------------------- | ------------------------- |
| 分句     | `split_text.py`                                          | 支援中英分句                    |
| 語言判斷   | `lang_detect.py`                                         | 判斷文章語言，套用 Prompt          |
| 清潔     | `clean_text.py`                                          | 清除 HTML 、空白、特殊字元          |
| AI 分析  | `risk_analyzer.py`                                       | 輸入句子、套用 Prompt、呼叫 GPT API |
| Prompt | `prompts/prompt_template_zh.txt`, `prompt_test_cases.md` | 依照區分系 Prompt              |
| 資料     | `data/labeled/initial_examples.csv`                      | 你已建立的 10 條風險範例            |
| 結果組合   | `result_aggregator.py`, `report_generator.py`            | 合成 JSON/表格/採用 Markdown    |
| 用戶回饋   | `feedback_handler.py`, `data/feedback/`                  | 收集使用者表示正確/錯誤              |

## 🔍 我們會怎麼訓練 AI？

1. 先建立一批真實條款（至50條）並手動標準風險類別
2. 用這些當作 AI 的 Prompt 範例模糊
3. 每次分析新條款時，用同樣格式輸出（等級、類型、理由）
4. 收集用戶回饋，作為未來標記資料

## 📅 開發階段計畫（預定）

### 第一階段：Chrome 外掛 MVP

* 使用者在看條款網頁時，點一下外掛
* AI 分析完全條款，用色範标示風險

### 第二階段：實製 AI 分析與回饋機制

* 關鍵字 + 句型分析 + Prompt
* 允許使用者指出 AI 判斷是否正確

### 第三階段：Web App + API

* 支援專線企業系統整合
* 支援 PDF / Word / txt 條款分析

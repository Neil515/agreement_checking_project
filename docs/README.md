# AI 合約條款風險分析工具

這是一個基於 GPT 模型的合約條款風險辨識工具，用於分析中英文合約條文，自動辨識「須注意」條款，輸出 JSON 結果。支援條文斷句、風險分類、主類別對應、雙語標註與視覺化分析，適用於風險預警、法律審閱輔助與合約教育場域。

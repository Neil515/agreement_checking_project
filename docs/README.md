# AI 合約條款風險分析工具

這是一個基於 GPT 模型的自動合約條款風險偵測工具，協助使用者快速分析中英文合約條文，辨識潛在須注意條款，並以 JSON 格式回傳結果。支援語言辨識、分類標註、前後端可切換顯示中英文結果，適合風險預警與視覺化呈現。

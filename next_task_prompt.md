## 🚀 明天可以接著做的任務

### ✅ 任務名稱：建立 Prompt 測試檔 + 展示標準解析格式

🔹 任務內容：

1. 新增一個文件 `prompt_test_cases.md`
2. 每一條含一篇條款，連同一段 Prompt 指示：

```
請判斷以下條款的風險等級（高、中、低），並說明理由與類型。
條款：「...內容...」
```

3. 運用你已建立的 10 條標準條款，各自編入 Prompt
4. 確保格式簡單、每條一段，便於之後自動測試

📄 結果示例：

```
請判斷以下條款的風險等級（高、中、低），並說明理由與類型。
條款：「本公司可隨時修改服務條款，且無須通知使用者。」
```

🔹 評估任務時間：30 分鐘－1 小時，先把測試檔做好，不用親身跟 AI 測試

---
